# skin-disease-predictor



A machine learning model that predicts skin disease type 

based on patient clinical and histopathological data.



\## 🎯 What it Predicts

6 different skin diseases:

\- Psoriasis

\- Seborrheic Dermatitis

\- Lichen Planus

\- Pityriasis Rosea

\- Chronic Dermatitis

\- Pityriasis Rubra Pilaris



\## 📊 Model Performance

| Model | Accuracy |

|---|---|

| Random Forest | 98.65% |



\## 🛠️ Built With

\- Python

\- Scikit-learn

\- Pandas

\- NumPy
